# Etwings

Etwings API wrapper for ruby.

## Installation

*Note: This gem is not yet published to rubygems.*

Add this line to your application's Gemfile:

    gem 'etwings'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install etwings
